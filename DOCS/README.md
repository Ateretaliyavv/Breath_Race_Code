# Breath-Race 
--עטרת ון לואן, טליה ולרשטיין, טל פלקובסקי, נועה אלבז, נועם משילקר--

**משחק נשימתי מאתגר וממכר, האם תצליח לנשום עד קו הסיום?!**

## מהות המשחק

משחק מירוץ עם רמות אתגר שונות, על שחקן לעבור מעל תהומות ביצירת גשרים ע"י נשיפה באורך המתאים, וקפיצה על מדרגות ע"י חוזק מתאים של נשיפה. בנשיפה לא מתאימה ייפול\יתקע השחקן, יפסל ויוכל להתחיל את השלב מחדש. משחק זה נועד לתרגול נשימות באורכים שונים וחוזק משתנה.
המשחק מיועד למחשב רגיל ולטלפון נייד.
